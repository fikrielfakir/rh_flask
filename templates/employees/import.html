Instructions
Format de fichier requis:
  • Format CSV (UTF-8)
  • Première ligne: en-têtes
  • Données séparées par virgule
Colonnes requises:
  • ID Employé - Unique
  • Nom - Nom complet
  • Email - Unique
  • Branche - Doit exister
  • Département - Doit exister
Formats de date:

Utilisez le format JJ/MM/AAAA
Exemple: 15/03/1990

Important: Les branches et départements doivent déjà exister dans le système.
Exemple de structure CSV
ID Employé Nom CIN Email Branche Département Date d'embauche Salaire
EMP001 Ahmed Alami AB123456 ahmed.alami@ceramica.ma Casablanca Ressources Humaines 01/01/2024 8000

Cette table montre seulement quelques colonnes principales. Le fichier complet contient toutes les informations d'employé.
